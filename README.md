# E-Learning Platform

The E-Learning Platform enables instructors to publish courses and learners to enroll, purchase, and track progress in an intuitive and scalable environment.

It demonstrates real-world full-stack development using Django, combining backend logic, frontend rendering, authentication, and third-party integrations.

## Key Features
__User Authentication & Authorization__

User registration and login

Role-based access (Student / Instructor / Admin)

Secure password hashing & session management

Profile management

## Course Management

Create, update, and delete courses

Upload course content (videos, PDFs, lessons)

Categorized courses for easy discovery

Course detail pages with descriptions & pricing

## Payment Integration

Secure online payment gateway integration

Paid and free courses support

Enrollment after successful payment

Transaction records for users and admin

## Student Dashboard

Enrolled courses overview

Learning progress tracking

Purchase history

Personalized recommendations

## Admin Dashboard

Platform analytics and reports
